# **App Name**: ResumeRight

## Core Features:

- Resume Upload: Accept resume uploads in PDF or DOCX formats.
- AI Resume Analysis: Use an AI tool to evaluate resumes for content, grammar, and formatting based on industry standards and best practices.
- Job Description Matching: Provide an estimated score representing the compatibility between a given resume and a job description. The higher the score, the better the potential match. The matching criteria may be explicit and/or implicit based on past successful candidates.
- Improvement Suggestions: Generate specific, actionable recommendations for improvement based on the AI resume analysis.
- Visualized Feedback: Visualize analyzed feedback using clear, concise displays to present evaluation summaries, metrics and scoring. Each area being evaluated should also be shown with specific textual feedback and actionable improvements

## Style Guidelines:

- Primary color: Professional blue (#2962FF) to inspire trust and clarity.
- Background color: Clean, desaturated light blue (#E6F0FF) for a professional and calm feel.
- Accent color: Muted green (#90EE90) to highlight positive feedback and actionable suggestions, ensuring visibility without being distracting.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text.
- Use minimalist, professional icons to represent different feedback categories (e.g., content, grammar, formatting).
- Employ a clean, structured layout with clear sections for resume uploading, feedback display, compatibility scores, and improvement suggestions.
- Incorporate subtle animations during the resume analysis process to enhance user engagement and provide feedback.
